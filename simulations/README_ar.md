# المحاكاة

## تسجيل مخاطر SAI ومقارنة السيناريوهات المفاهيمية

يحتوي هذا الدليل على نموذج محاكاة مبسط لمقارنة ملفات المخاطر في سيناريوهات حقن الهباء الجوي في الستراتوسفير (SAI).

هذا ليس نموذجًا مناخيًا.

إنه أداة تسجيل شفافة لتوضيح أن SAI لا ينبغي تقييمه فقط من خلال تقليل ضوء الشمس أو انخفاض متوسط درجة الحرارة العالمية.

---

## الملفات

- [sai_risk_simulation.py](sai_risk_simulation.py)  
  سكربت Python يحسب درجات المخاطر المرجحة لعدة سيناريوهات SAI.

- [sai_risk_simulation_results.csv](sai_risk_simulation_results.csv)  
  ملف CSV يحتوي على نتائج السيناريوهات الافتراضية.

---

## محاور التقييم

```text
R1 حمل الجسيمات الجوية الموجود مسبقًا
R2 عدم اليقين في طبقات الهباء الرأسية
R3 ضعف الترسيب الرطب
R4 فقدان التثبيت السطحي
R5 خطر إعادة رفع الجسيمات
R6 اضطراب السحب والهطول
R7 خطر التفاعل مع الحرارة الخارجة والأشعة تحت الحمراء
R8 تضرر التغذية الراجعة الطبيعية للتبريد
R9 خطر الحوكمة والنزاع الإقليمي
R10 خطر صدمة التوقف
```

تُسجل كل فئة من 0.0 إلى 1.0.

---

## طريقة التشغيل

```bash
python sai_risk_simulation.py
```

يعرض السكربت النتائج في الطرفية ويكتب ملف CSV التالي:

```text
sai_risk_simulation_results.csv
```
